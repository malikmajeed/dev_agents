# Environment Variables Registry

Managed by DevAgent. **Never put real secret values in this file.**

| Key | Description | Scope | Status | Feature |
|-----|-------------|-------|--------|---------|
| NEXT_PUBLIC_API_BASE_URL | Base URL of the backend API (e.g., https://api.example.com) used by the frontend axios instance | frontend | pending | Donor management |
| EMAIL_USER | Gmail address used to send receipt emails | backend | pending | Donation receipt emails |
| EMAIL_PASS | App password or OAuth token for the Gmail account | backend | pending | Donation receipt emails |
